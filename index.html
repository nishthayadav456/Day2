Q1. Explain Br tags?

Ans. The br tag in HTML documents is used to create a line break in a text. It is an empty tag which means it does not need an end tag .
It is generally used in poems and addresses where the division of line is necessary.

Q2.Explain how many different types of lists we have in HTML?

Ans. There are three types of lists that is ordered list , unordered list and description list.
ordered list- It starts with the ol tag and ends with the /ol tag. The list items start with the li tag and end with /li tag.
unordered list-An unordered list starts with the ul tag. Each list item starts with the li tag. The list items will be marked with bullets (small black circles) by default.
Description list-A description list is a list of terms, with a description of each term. The dl tag defines the description list, the dt tag defines the term (name), and the dd tag describes each term.
